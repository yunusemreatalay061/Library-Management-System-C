# Library Management System 📚

## About the Project
This is a console-based Library Management System developed entirely in C. The primary goal of this project is to demonstrate a strong understanding of fundamental computer science concepts, low-level memory management, and structured data handling. It allows users to efficiently organize, track, and manage a collection of books.

## Key Features
* **Dynamic Memory Allocation:** Extensively used `malloc` and `free` to dynamically allocate memory for storing book records, ensuring optimal resource usage and preventing memory leaks.
* **Pointer & Array Mastery:** Deeply integrated pointer-array relationships for fast, direct memory access and efficient data manipulation.
* **Structured Data:** Utilized C `structs` to bundle related book attributes together (Title, Author, Publication Year, and Price) for clean and organized data handling.
* **Algorithmic Logic:** Implemented core programming logic and control flows to manage user inputs and system operations seamlessly.

## Technologies & Core Concepts
* **Programming Language:** C
* **Core Concepts:** Pointers, Dynamic Memory Allocation (malloc/free), Memory Management, Structs, Procedural Programming

## What I Learned
Building this system took my foundational engineering skills to the next level. It provided hands-on, critical experience in managing how data is stored, accessed, and safely freed in memory using pointers and dynamic allocation—skills that are essential for understanding complex Data Structures and system-level programming.
